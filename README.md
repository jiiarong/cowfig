# cowfig
A config/config.json generator


## Help

``` bash
$ node cowfig.js -h
Usage: ./bin/cowfig [-t TEMPLATE_PATH] [-s RESOURCE_PATH] [-d DESTATION_PATH] [-e ENV]

```


## Example

```
$ cd util/cowfig
$ node ./bin/cowfig -t ../../config-source/template/ -s ../../config-source/source/ -d ../../ 
```
